# torce

# Setup

1. .env ファイルを作成

```
touch .env
```

2. .env.example をコピペして環境変数を埋めて保存

```.env
FIREBASE_API_KEY=xxx
FIREBASE_AUTH_DOMAIN=xxx
FIREBASE_PROJECT_ID=xxx
FIREBASE_STORAGE_BUCKET=xxx
FIREBASE_MESSAGING_SENDER_ID=xxx
FIREBASE_APP_ID=xxx
```

3. node_modules のインストール

```
npm i
```

# Run

```
npm run start:dev
```
